I. RTBD Multi-Tenant Arbitration

Request Isolation

Each KV block is tagged with a 16-bit Request_ID in hardware. This ensures that even if Request A and Request B share the same layer or head indices, their data is physically isolated in memory.

The Prefix Exception

Request_ID 0x0000 is reserved for shared context (system prompts). Hardware reference counters prevent eviction as long as any active request depends on it.

II. Hardware CAM Match-Line Logic

To achieve single-cycle lookup across 65k entries, the RTBD uses a parallel comparator array:

SRAM Core + 4T-Comparator Match Cell

Match = (In_ID == Tag_ID) && (In_Layer == Tag_Layer) && (In_Token == Tag_Token)

HIT: Resolve Phys_Addr & Tier_Loc | MISS: Trigger HEPC Prefetch
