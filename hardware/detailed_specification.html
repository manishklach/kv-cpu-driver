2. HEPC Priority Logic (SystemVerilog)

always_comb begin
  // Priority = wR*R + wF*F + wS*S + wD*D
  p_score = (w_r * R) + (w_f * F) + (w_s * S) + (w_d * D);

  // Autonomous Staging Decision
  evict_req = (p_score < EVICT_T) && !is_guarded_prefix;
  stage_req = (p_score > PREFETCH_T);
end

3. The Scoring Formula[cite: 1]

$$P(B_i) = w_r \cdot R + w_f \cdot F + w_s \cdot S + w_d \cdot D \pmod{2^{16}}$$
  • w_s * S: The "Step Proximity" component that makes the hardware aware of the LLM decode step[cite: 1].
  • w_d * D: The hardware-level prefix guard that prevents OOM during shared prompt multi-tenancy[cite: 1].
