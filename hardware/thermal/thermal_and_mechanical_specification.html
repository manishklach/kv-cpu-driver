1. NMCE DPU Array (RTL Logic)

The SystemVerilog core for the Near-Memory Compute Engine (Quadrant A):

// 128-wide Systolic MAC Unit
always_ff @(posedge clk) begin
  for (int i = 0; i < B; i++) begin
    dot_prod = 0;
    for (int j = 0; j < D; j++) begin
      dot_prod += query[j] * key_sram[i][j];
    end
    out_score[i] = piecewise_exp(dot_prod * scale);
  end
end

3. Autonomous Decision Logic

Hardware-enforced priority scoring for KV-block migration[cite: 1]:

$$P(B_i) = w_r \cdot R + w_f \cdot F + w_s \cdot S + w_d \cdot D \pmod{2^{16}}$$

[Step-Advance] -> Recompute Priority -> [Compare Thresh] -> [Async DMA Trigger]

  • IF P(Bi) < Evict_T: Evict to T2/T3[cite: 1]
  • IF P(Bi) > Prefetch_T: Stage to T1[cite: 1]
  • IF Prefix_Flag=1: Guard in T1[cite: 1]
