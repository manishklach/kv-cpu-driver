Quadrant Specification

QUADRANT A: NMCE
  • DPU Array: 128x MAC units, 16-bit FP/BF[cite: 1]
  • SRAM: 16KB Query Buffer (64 slots)[cite: 1]
  • Engine: Piecewise exp() approx unit[cite: 1]
QUADRANT B: RTBD
  • CAM: 65,536-entry Hardware Tag Store[cite: 1]
  • Logic: Single-cycle parallel comparator[cite: 1]
  • Tracking: Request, Layer, Head, Token[cite: 1]

Multi-Tier Management Strategy

Tier Medium Latency Role in Closed Loop[cite: 1]
T0 GPU HBM3e ~100 ns Hot tokens being computed in current layer.
T1 KV-CPU (LPDDR5X) ~150 ns Hardware-managed staging & scalar scoring.
T2 Host DDR5 ~400 ns Evicted warm blocks; addressable via CXL.mem.
T3 NVMe PCIe 5.0 ~100 µs Cold context persistence / session archive.
